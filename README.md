# MNIST-In-C
Throwing Pytorch to the Curb

Tackling MNIST with the standard library in C. <br/>
Best accuracy: _95.65%_

Clone the repository and download [MNIST](http://yann.lecun.com/exdb/mnist/) into:
- `res/train_images.gz`
- `res/train_labels.gz`
- `res/test_images.gz`
- `res/test_labels.gz`

Build and run with `make`
